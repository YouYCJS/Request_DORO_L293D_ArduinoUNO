# Request-bluetooth-rc-l293d-arduino 
# Bluetooth RC Motor Controller (블루투스 무선 모터 구동 회로)

> HC-06 블루투스로 명령을 받아 L293D 모터 드라이버로
> 좌/우 모터를 제어하는 무선 조종 회로. Altium 독학 후
> 처음 설계한 PCB 프로젝트입니다.



## 개요
스마트폰에서 블루투스로 보낸 명령을 HC-06이 수신,
아두이노가 이를 해석해 L293D를 통해 두 개의 DC 모터를
정/역회전시켜 전진·후진·좌회전·우회전을 구현합니다.

## 회로 구성
- **MCU**: Arduino UNO
- **무선 통신**: HC-06 (블루투스, UART D0/D1)
- **모터 드라이버**: L293D (듀얼 H-브리지)
- **구동부**: DC 모터 ×2 (left / right)
- **전원**: 로직(아두이노)과 모터(Battery) 전원 분리

## 신호 흐름
블루투스 명령 → HC-06(RX/TX) → 아두이노 UART →
L293D ENABLE/INPUT 제어 → 모터 정·역회전

## 배운 점 / 회고
- Altium 첫 설계작 — 스키매틱 작성, 라이브러리/심볼,
  네트 연결의 기본기를 익힌 프로젝트
- 로직 전원과 모터 전원을 분리해야 하는 이유를 체득
- (이후 V1~V2 모터 드라이버 보드 설계로 발전)
